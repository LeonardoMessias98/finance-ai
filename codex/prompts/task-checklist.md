# Task Checklist

Antes de encerrar uma tarefa relevante, confirme:

- li `AGENTS.md`
- li todos os arquivos em `codex/contexts`
- mantive a organização por feature
- não misturei UI, regra de negócio e acesso ao banco
- mantive a conexão MongoDB centralizada
- validei entradas com Zod quando havia entradas externas
- usei Server Actions quando a tarefa exigia mutação
- mantive os arquivos pequenos e explícitos
- atualizei a documentação necessária
- registrei a tarefa em `codex/history`
