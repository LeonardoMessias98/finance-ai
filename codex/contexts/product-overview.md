# Product Overview

## Produto

`finance-ai` é uma aplicação de gestão financeira pessoal desenhada para crescer com suporte de agentes de IA desde o início.

## Objetivos do bootstrap

- iniciar com base simples e estável
- reduzir tempo de setup para novas features
- facilitar leitura e manutenção por outros agentes
- evitar retrabalho arquitetural nas primeiras entregas

## Direção inicial

- começar com dashboard vazio e layout base
- adicionar features por incrementos pequenos
- manter foco em clareza de domínio antes de automações avançadas
