# Business Rules

## Regras atuais

- o bootstrap não implementa regras financeiras de negócio ainda
- o dashboard inicial deve permanecer vazio do ponto de vista funcional
- a aplicação deve estar pronta para receber módulos de receitas, despesas e categorias

## Regras para próximas features

- toda entrada externa deve ser validada
- persistência deve passar por `repositories`
- regra de negócio deve passar por `services`
- UI não deve conhecer detalhes de banco
