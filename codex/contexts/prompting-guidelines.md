# Prompting Guidelines

## Diretrizes para agentes

- leia `AGENTS.md` antes de qualquer alteração
- leia os arquivos de `codex/contexts` antes de propor estrutura nova
- preserve o padrão do projeto antes de introduzir melhorias
- prefira mudanças pequenas, explícitas e fáceis de revisar

## Entrega

- liste arquivos criados
- liste arquivos alterados
- explique decisões técnicas de forma objetiva
- registre tarefas importantes em `codex/history`
